                      GENERATE MORE PUZZLES

EASY WAY (Windows):
-------------------
Double-click: quick-start.bat
Choose option from menu

COMMAND LINE:
-------------
python generate.py --start 2025-01-01 --days 30 --out site --difficulty easy --categories general,tech,travel,kids,devotional,holiday

PARAMETERS:
-----------
--start      Start date (YYYY-MM-DD)
--days       Number of days to generate
--out        Output folder (default: site)
--difficulty easy, medium, or hard
--categories Comma-separated list

EXAMPLES:
---------
# Generate 7 days
python generate.py --start 2025-01-01 --days 7 --out site

# Generate 90 days (3 months)
python generate.py --start 2025-01-01 --days 90 --out site

# Generate with medium difficulty
python generate.py --start 2025-01-01 --days 30 --out site --difficulty medium

================================================================================
                        TROUBLESHOOTING
================================================================================

SITE NOT LOADING:
-----------------
1. Check DNS propagation: dnschecker.org
2. Clear browser cache: Ctrl+Shift+R
3. Wait 24 hours for DNS
4. Check Netlify deployment logs

PUZZLES NOT GENERATING:
-----------------------
1. Check Python version: python --version (need 3.7+)
2. Read error message
3. Try with --days 1 first
4. Check generate.py for syntax errors

GITHUB ACTIONS NOT RUNNING:
---------------------------
1. Check Actions tab in GitHub
2. Verify file location: .github/workflows/generate.yml
3. Check permissions: Settings ‚Üí Actions ‚Üí Read and write
4. Manually trigger: Actions ‚Üí Run workflow
