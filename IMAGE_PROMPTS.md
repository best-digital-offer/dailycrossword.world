## Brand Color Palette

### Primary Colors
- **Electric Blue:** #1e90ff (Primary brand color)
- **Deep Purple:** #764ba2 (Secondary brand color)
- **Gradient:** Linear gradient from #667eea to #764ba2

### Secondary Colors
- **Success Green:** #4caf50 (Easy difficulty, correct answers)
- **Warning Orange:** #ff9800 (Medium difficulty)
- **Error Red:** #f44336 (Hard difficulty, incorrect answers)
- **Info Blue:** #2196f3 (Hints, information)

### Neutral Colors
- **Dark Gray:** #333333 (Text, borders)
- **Medium Gray:** #666666 (Secondary text)
- **Light Gray:** #f7f7fb (Background)
- **White:** #ffffff (Cards, puzzle grid)

### Usage Guidelines
- Use gradient for hero sections and CTAs
- Use primary blue for links and interactive elements
- Use neutral colors for text and backgrounds
- Use secondary colors sparingly for emphasis

## Typography Recommendations

- **Font:** Poppins, Montserrat, or Inter (Bold)

- **Line Height:** 1.6
- **Color:** #333333

### Puzzle Grid
- **Font:** Monospace or Arial Bold
- **Size:** 20px (desktop), 16px (mobile)
- **Color:** #000000

## Accessibility Considerations

### Image Alt Text Examples
```
Logo: "Daily Crossword logo - Free daily crossword puzzles"
Hero: "People enjoying crossword puzzles on various devices"
Category icon: "Technology category icon - laptop symbol"
Difficulty badge: "Easy difficulty badge - one star"
```

### Color Contrast
- Ensure text has 4.5:1 contrast ratio minimum
- Use WCAG AA compliant color combinations
- Provide alternative text for all images
- Don't rely on color alone to convey information

## Image Optimization

### Web Optimization
- **Format:** WebP (with PNG/JPG fallback)
- **Compression:** TinyPNG or ImageOptim
- **Max file size:** 100KB for hero images, 50KB for icons
- **Lazy loading:** Implement for below-fold images

### Responsive Images
```html
<picture>
  <source srcset="hero-large.webp" media="(min-width: 1200px)" type="image/webp">
  <source srcset="hero-medium.webp" media="(min-width: 768px)" type="image/webp">
  <source srcset="hero-small.webp" media="(min-width: 320px)" type="image/webp">
```

## Legal Considerations

### Copyright
- Use AI-generated images (you own rights)
- Or purchase stock photos with commercial license
- Avoid copyrighted puzzle designs from competitors
- Create original illustrations

### Attribution
- AI-generated images typically don't require attribution
- Stock photos may require attribution (check license)
- Custom illustrations - negotiate rights with designer

### Model Releases
- Not needed for AI-generated people
- Needed for photos of real people (if using stock)
